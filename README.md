# Nicolas Vazquez Pizzi - Portfolio

Sitio web profesional de Nicolas Vazquez Pizzi, especialista en mercados financieros argentinos y desarrollo de soluciones tecnológicas.

## Ver sitio

**[https://nicomvazquez.github.io/nicomvazquez-web/](https://nicomvazquez.github.io/nicomvazquez-web/)**

## Tecnologías

- React 19
- Vite 7
- Tailwind CSS 4
- React Icons

## Desarrollo local

```bash
npm install
npm run dev
```

## Despliegue

El sitio se despliega automáticamente en GitHub Pages cuando se hace push a la rama `master`.
